# Cursor Instructions: Pacific Coast Title Website

## Project Context

This is a website redesign for Pacific Coast Title Company (pct.com), a California-based title and escrow company. The initial scaffolding was generated by V0 and deployed to Vercel. Your job is to build out the complete site based on the specifications below.

**Live V0 Prototype**: https://v0-pacific-coast-title-redesign.vercel.app/
**Current Production Site**: https://www.pct.com (reference for content/data)
**Tech Stack**: Next.js 14+ (App Router), React, Tailwind CSS, TypeScript

### Design Principles
- Clean, professional, trustworthy (financial services aesthetic)
- Generous whitespace
- Subtle animations (fade-in on scroll, hover states)
- Mobile-first responsive design
- Avoid over-styling; let content breathe

## External Integrations

### TESSA Chat Widget
- The current site has a TESSA AI chat widget
- For now, create a placeholder floating button that opens a modal
- Style it with the TESSA branding (sparkle icon)
- Position: fixed bottom-right

### Google Maps
- Each location page needs an embedded map
- Use @react-google-maps/api or an iframe embed
- Include "Get Directions" link

### Forms
- Contact forms should POST to a backend endpoint (or use Vercel serverless functions)
- Consider Resend or SendGrid for email notifications
- Add form validation with react-hook-form + zod

## Testing Checklist

Before considering a page complete:

- [ ] Mobile responsive (test at 375px, 768px, 1024px, 1440px)
- [ ] All links work (no 404s)
- [ ] Images have alt text
- [ ] Page has proper metadata
- [ ] Forms validate and submit correctly
- [ ] Animations don't cause layout shift
- [ ] Lighthouse score > 90 (Performance, Accessibility, SEO)
- [ ] Contact info is correct (not placeholder)
